# ISIDORITO

Aplicacion para gestion de negocio minorista/mayorista con caja, stock, precios y administracion operativa.

## Inicio rapido
- Instalar dependencias: `npm install`
- Desarrollo: `npm run dev`
- Produccion local: `npm start`

## Punto de entrada
- Servidor principal: `index.js`

## Estructura de documentacion
- Documentacion operativa vigente: `docs/operativo/`
- Documentacion historica y reportes: `docs/archivo/`

## Documentos clave
- Resumen funcional y estado actual: `docs/operativo/RESUMEN_FUNCIONAL_Y_ESTADO.md`
- Politica de limpieza documental: `docs/operativo/POLITICA_DOCUMENTACION.md`

## Nota
- `app.js` no es el servidor principal actual; se usa `index.js`.
