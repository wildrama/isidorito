┌───────────────────────────────────────────────────────────────────────────────┐
│ 🔍 PRUEBA CON CÓDIGO DE BARRA                                               │
└───────────────────────────────────────────────────────────────────────────────┘

OPCIÓN 1: Emular con teclado
  1. Ir a: /administrador/stock/buscar
  2. Pegar código: 7791234567890 (o el que tengas)
  3. Sistema detecta automáticamente como barcode
  4. Muestra resultado exacto

OPCIÓN 2: Con Reader Real (USB)
  1. Conectar reader USB
  2. Ir a: /administrador/stock/buscar
  3. Hacer focus en input
  4. Hacer scan
  5. Sistema busca automáticamente

┌───────────────────────────────────────────────────────────────────────────────┐
│ ❓ TROUBLESHOOTING                                                           │
└───────────────────────────────────────────────────────────────────────────────┘

❌ "Error: /administrador/stock/buscar no existe"
   → npm start (verificar que servidor está corriendo)

❌ "Búsqueda no devuelve resultados"
   → Verificar que hay productos en MongoDB
   → Ver console F12 → Network tab

❌ "Barcode no se detecta"
   → Verificar que código tiene al menos 3 dígitos
   → Input debe tener focus
   → Ver console.log en F12

❌ "Producto individual no carga"
   → Verificar ID del producto en URL
   → Verificar que producto existe en MongoDB
