================================================================================
                              FLUJO DE USO
================================================================================

PASO 1: Buscar producto
  → Ir a http://localhost:3037/administrador/buscar
  → Ingresar nombre del producto
  → Click "Ver Detalle"

PASO 2: Editar precios
  → Click "💰 Editar Precios"
  → Se abre editPrecio.ejs con las 3 CARDS

PASO 3: Cambiar precio (opción 1 - %)
  → Ingresar 10 en "Aumento %"
  → "Nuevo Precio" se actualiza automáticamente: $110
  → Click "✅ Guardar Cambio"

PASO 3: Cambiar precio (opción 2 - Manual)
  → Ingresar 125.50 en "Aumento Manual"
  → "Nuevo Precio" se actualiza automáticamente: $125.50
  → Click "✅ Guardar Cambio"

PASO 4: Guardar
  → Botón cambia a "⏳ Guardando..."
  → Backend procesa la actualización
  → CARD hace flash verde
  → Mensaje: "✅ Precio actualizado: $110.00 (+$10.00)"

PASO 5: Sincronización
  → Cambio se guarda en localStorage
  → Cambio se envía al backend (PUT)
  → Base de datos se actualiza
  → Mensaje desaparece después de 4 segundos

PASO 6: Volver
  → Click "← Volver al Detalle"
  → priceSync.js detecta cambio en localStorage
  → stockIndividual.ejs se actualiza automáticamente
  → Precio muestra valor nuevo con animación

================================================================================
                         SINCRONIZACIÓN TÉCNICA
================================================================================

FLUJO DE SINCRONIZACIÓN:

1. USUARIO EDITA PRECIO en editPrecio.ejs
   └─ editPrice.js calcula nuevo valor
   └─ Preview se muestra en "Nuevo Precio"

2. USUARIO CLICK "Guardar"
   └─ Validación local (no negativo, es número)
   └─ Envía PUT a /administrador/productos/:id/precmin
   └─ Backend valida (repite validación)
   └─ Mongo actualiza documento
   └─ Backend devuelve JSON con éxito

3. GUARDAR EN LOCALSTORAGE
   └─ editPrice.js guarda en localStorage:
      {
        productId: "5f7a8c1d...",
        precioMinorista: 110.00,
        timestamp: "2025-11-19T..."
      }

4. USUARIO VUELVE A DETALLE
   └─ Navega a /administrador/productos/:id
   └─ stockIndividual.ejs se carga
   └─ priceSync.js se ejecuta automáticamente

5. SINCRONIZACIÓN AUTOMÁTICA
   └─ priceSync.js detecta localStorage
   └─ Obtiene productId de la URL
   └─ Actualiza elemento DOM con animación
   └─ Limpia localStorage
   └─ Inicia sincronización backend cada 10s

6. BACKEND SYNC (cada 10 segundos)
   └─ GET /administrador/productos/:id/precios
   └─ Obtiene precios actuales de MongoDB
   └─ Compara con valores mostrados
   └─ Actualiza si hay diferencias
   └─ Verifica multi-dispositivo

================================================================================
                            TROUBLESHOOTING
================================================================================

PROBLEMA: Los precios no se sincronizan
SOLUCIÓN:
  1. Verifica que Axios está cargado (F12 → Network)
  2. Verifica que localStorage está habilitado
  3. Recarga la página
  4. Espera 10 segundos para sync de backend

PROBLEMA: Las animaciones no aparecen
SOLUCIÓN:
  1. Limpia caché: Ctrl+Shift+Delete
  2. Recarga: Ctrl+R
  3. Abre F12 → Console, busca errores

PROBLEMA: El precio no se guarda en BD
SOLUCIÓN:
  1. Verifica que ingresaste número positivo
  2. Abre F12 → Console, mira error exacto
  3. Verifica que usuario es ADMINISTRADOR
  4. Intenta de nuevo

PROBLEMA: No veo los logs en console
SOLUCIÓN:
  1. Abre F12 (Developer Tools)
  2. Ve a pestaña "Console"
  3. Busca por "[editPrice.js]" o "[priceSync.js]"
  4. Si no hay nada, recarga página
