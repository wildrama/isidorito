POST /api/search/smart (RECOMENDADO)
  ├── Body: { query: "coca" }
  ├── Auto-detecta: barcode vs texto
  └── Response: Array de productos

POST /api/search/productos
  ├── Body: { query: "coca", sort: "relevancia", limit: 50 }
  └── Response: Búsqueda por nombre, marca, código

POST /api/search/barcode
  ├── Body: { barcode: "7791234567890" }
  └── Response: Match exacto de código

POST /api/search/advanced
  ├── Body: { query: "coca", precioMin: 1, precioMax: 5 }
  └── Response: Búsqueda multi-criterio
